Software Development

Dev Style

I ❤️ ❤️ ❤️ Python and try to write beautiful and easy-to-maintain code, often using object oriented programming. I sometimes use Cython/C for speed, if multi-threading is not enough. I know my way in R, but thoroughly despise it. I use Bash all the time, though I would never consider it for a script longer than a few lines. Let's face it, it is ugly as hell.

I do all my development on Linux/Ubuntu and my production code runs on a Linux/RedHat high performance computing cluster. I don't really like windows and OSX but since other people do I do my best to write portable multi-platform code. I have experience in GPU computing and containerisation and I am getting up to speed in cloud computing as well.

Over the last 7 years I have been actively developing a number of scripts, APIs and analytic workflows for bioinformatic analyses. In particular, I wrote several tools to analyse Oxford Nanopore sequencing data, some of which have been gaining popularity in the community
